## To-Do list
A project where the user puts a task that they wanna do and after they finish it they can mark it as done and then delete it!

## Feautres
- Freedom to add any task
- can mark task done
- can delete task
- lets the user know if the task is already marked done
- marks the task as undone by default

## How to Run:
1. Insall python
2. Run:
```bash
python to-do-list.py
```
